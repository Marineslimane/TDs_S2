# Réponses aux questions

## Exercice 3

1. On constate que l'algorithme utilisé pour le tri de la librairie standard est plus rapide que le bubble sort et le merge sort en général. On remarque aussi que pour des tailles d'array petites, le bubble sort est en général plus rapide que le merge sort et que celui de la librairie standard. Sinon, le merge sort est plus rapide que le bubble sort pour des plus grandes tailles d'array.
<br>   
2. En effet, après recherche, la complexité dans le pire des cas pour le tri de la librairie standard est en  $\mathcal{O}(n \log n)$ alors que celle du bubble sort est de $\mathcal{O}(n^2)$ et celle du merge sort est de $\mathcal{O}(n \log n)$, ce qui explique ces différences de temps.
